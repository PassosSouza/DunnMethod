# CV Contribution Analysis: Dunn Method

A Python toolkit for separating capacitive and faradaic (diffusional) contributions in cyclic voltammetry (CV) data using the **Dunn method**, with synthetic ideal data generated via the **Butler–Volmer equation**.

---

## Overview

This repository provides a complete pipeline to:

1. **Generate ideal CV data** (`Generate_Ideal_Data.ipynb`) — Simulate cyclic voltammograms at multiple scan rates using the Butler–Volmer formalism. The output includes the total current as well as the separated **capacitive** and **faradaic** contributions, which serve as a ground-truth benchmark.

2. **Separate contributions experimentally** (`DunnMethod`) — Apply the Dunn method to real or synthetic CV data acquired at different scan rates to deconvolute the total current into:
   - **Capacitive current** ($i_c = k_1 \nu$)
   - **Faradaic/diffusional current** ($i_f = k_2 \sqrt{\nu}$)

---


### File Descriptions

| File / Folder | Description |
|---------------|-------------|
| `Generate_Ideal_Data.ipynb` | Jupyter notebook that generates synthetic CV data using the Butler–Volmer equation. Produces total current, capacitive current (`ic`), and faradaic current (`if`) for various scan rates. |
| `DunnMethod` | Python script or notebook implementing the Dunn separation method. Fits $i(V, \nu) = k_1 \nu + k_2 \sqrt{\nu}$ at each potential and returns the separated contributions plus standard deviations. |
| `Data/` | Directory containing the input CV data files. Each file should have columns for **Potential (V)** and **Current (A)** (or similar). |
| `Data/ideal/` | Subdirectory containing the benchmark ideal data generated by `Generate_Ideal_Data.ipynb`. Used to validate the accuracy of the Dunn separation. |

## Theory

### Butler–Volmer (Ideal Data Generation)
The Butler–Volmer equation describes the electrode kinetics for a single-electron transfer reaction:

$$i = i_0 \left[ \exp\left(\frac{\alpha_a n F \eta}{RT}\right) - \exp\left(-\frac{\alpha_c n F \eta}{RT}\right) \right]$$

where $\eta = E - E^0$ is the overpotential. In the notebook, this is coupled with a capacitive background to mimic real-world CV behavior.

### Dunn Method (Contribution Separation)
At any fixed potential $V$, the total current measured at scan rate $\nu$ is modeled as:

$$i(V, \nu) = k_1(V) \cdot \nu + k_2(V) \cdot \sqrt{\nu}$$

- $k_1 \nu$ → **Capacitive contribution** (surface-controlled, linear in scan rate)
- $k_2 \sqrt{\nu}$ → **Faradaic contribution** (diffusion-controlled, linear in square root of scan rate)

By measuring CVs at several scan rates and fitting the above equation at each potential, one can quantitatively separate the two contributions.

**Reference:**  
B. Dunn *et al.*, "Electrical Energy Storage for the Grid: A Battery of Choices," *Science*, **2011**, 334, 928–935.
